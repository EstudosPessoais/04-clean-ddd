# 04-clean-ddd

Um projeto utilizando DDD e Clean Arch

Projeto desenvolvido no curso da Rocketseat.


# Subdomínios

- Core: O que dá dinheiro
- Supporting: Suporte para o core funcionar
- Generic: Você precisa, mas não são muito importantes.


## Exemplos
## Core

- Compra
- Catálogo
- Pagamento
- Entrega

### Supporting

- Estoque

### Generic

- Notificação
- Promoções
- Chat
